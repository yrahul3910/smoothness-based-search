Round 3 — Software-Configuration Optimization

↑ All rounds · metric: distance-to-heaven · task: direct configuration optimization

Task

The data/optimize/config/ datasets are lookup tables of measured software configurations. Unlike Rounds 1–2 (which tuned a decision tree's hyperparameters and scored R²), this round does direct configuration optimization: a method reveals up to N configurations' performance and returns the best it found, scored by distance-to-heaven (d2h) — the Euclidean distance to the ideal point in normalized objective space (lower is better). This is the canonical moot/SE-config-tuning task that TPE is designed for.

Why the method set changed from Round 2. Because evaluating a config's d2h is the legitimate objective (there is no held-out test set to "peek" at), the fair/peek split collapses; and because each config has a single measured value with no cheap/expensive fidelity, BOHB's Hyperband has nothing to halve on. So the 9-method Round-2 set reduces to five: Random, TPE (SOTA optimizer), Greedy surrogate (ablation: no smoothness), INGS (smoothness-aware), and SMAS (smoothness-adaptive). SMAS's β is reinterpreted as the smoothness of the d2h response surface over evaluated configs, replacing the trained-learner β of Rounds 1–2.

Experiment

Exp 9 — Smoothness-guided configuration optimization (8 datasets × 50 repeats, budgets 10/20/30/50, with per-dataset convergence curves).

Headline. The smoothness-aware surrogate optimizer beats TPE at N=30 (SMAS 6W/0T/2L, INGS 5W/0T/3L), beats Random (6W/1T/1L), and beats the no-smoothness Greedy ablation (margin grows with budget: 1W/2T/5L at N=10 → 5W/2T/1L at N=50). Mean d2h at N=30: SMAS 0.051, INGS 0.052, TPE 0.094, Greedy 0.068, Random 0.083. This is the strongest result across all three rounds — unlike the regression task where TPE dominated, the smoothness prior wins here, and TPE only competes on the 39-dim SQL space.
Honest caveat on SMAS vs INGS. SMAS = INGS + an adaptive κ controller based on the surrogate tree's β-smoothness. At 50 repeats SMAS and INGS are statistically indistinguishable (SMAS vs INGS Mann-Whitney: 0W/7T/1L at N=30; paired Wilcoxon p > 0.1 at every budget). The visible separation on the convergence plots is within noise. So the win over TPE comes from the RBF + anti-Laplacian + UCB core — INGS — not from the β-controller. INGS is the simpler method that achieves the result; SMAS is its β-controlled variant we couldn't show adds significant value on this benchmark.
